⚠️ DOCKER NOT INSTALLED

To run the URBAN platform, you need PostgreSQL and Redis.

=== EASIEST SOLUTION: Install Docker Desktop ===

1. Download Docker Desktop for Windows:
   https://www.docker.com/products/docker-desktop/

2. Install and restart your computer

3. Open PowerShell and run:
   docker run -d --name urban-postgres -e POSTGRES_PASSWORD=postgres -e POSTGRES_DB=urban_db -p 5432:5432 postgres:14
   docker run -d --name urban-redis -p 6379:6379 redis:6

4. Continue with setup:
   npm run db:migrate
   npm run db:seed
   npm run dev

=== ALTERNATIVE: Install PostgreSQL & Redis Locally ===

See WINDOWS_SETUP.md for detailed instructions

PostgreSQL: https://www.postgresql.org/download/windows/
Redis: https://github.com/microsoftarchive/redis/releases

=== CLOUD ALTERNATIVE (No local install needed) ===

Use free cloud services:
- PostgreSQL: https://supabase.com (free tier)
- Redis: https://upstash.com (free tier)

Update your .env file with their connection strings.
